  * [[http://pism.github.io/uaf-iceflow/|ice sheet modeling at UAF]]
  * [[https://www.cresis.ku.edu/|CReSIS]] at the University of Kansas has radar data for bed elevation and ice thickness
  * [[http://www.nsidc.org/|NSIDC]] (National Snow and Ice Data Center) is a master source for data

===== other open source ice sheet models =====

  * the Community Ice Sheet Model is an open source ice sheet model which serves as the ice dynamics component of [[http://www2.cesm.ucar.edu/|CESM]]
  * [[http://www.sicopolis.net/|SICOPOLIS]](Simulation COde for POlythermal Ice Sheets) is a 3-d dynamic/thermodynamic model which simulates the evolution of large ice sheets
  * the [[http://elmerice.elmerfem.org/wiki/doku.php|ELMER ice sheet model]] is an open source multiphysics simulation package adapted to simulations of ice sheets and glaciers
  * [[http://issm.jpl.nasa.gov/|ISSM]] (Ice Sheet System Model) is an open source finite element model at NASA JPL which supports several stress balances
===== collaborations and intercomparisons =====

    * [[http://websrv.cs.umt.edu/isis/index.php/SeaRISE_Assessment|the SeaRISE assessment process]] in the US plans to contribute model-based estimates of century-scale sea level changes from ice sheet and ice shelf dynamics to the IPCC AR5 report in 2013 
    * [[http://www.ice2sea.eu/|ice2sea]] is a European effort which is somewhat comparable to SeaRISE
    * [[http://homepages.vub.ac.be/%7Ephuybrec/eismint.html|EISMINT (European Ice Sheet Modeling INiTiative)]] is a past intercomparison effort to test and compare existing numerical shallow ice-sheet, ice-shelf, and glacier models as they were run by several groups worldwide
    * [[http://homepages.vub.ac.be/%7Ephuybrec/ismip.html|ISMIP (Ice Sheet Model Intercomparison Project)]] continues intercomparison, including less-shallow and Stokes models (ISMIP-HOM) and rapid sliding events (ISMIP-HEINO)
    * [[http://homepages.ulb.ac.be/~fpattyn/mismip/welcome.html|MISMIP]] is an intercomparison of flowline marine ice sheet models
    * [[http://homepages.ulb.ac.be/~fpattyn/mismip3d/welcome.html|MISMIP3D]] is an [[http://www.ice2sea.eu/|ice2sea]]-sponsored intercomparison of three-dimensional marine ice sheet models
